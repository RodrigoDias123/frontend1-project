Funcionalidades

Tudo o que a sua equipa precisa

Quadro Kanban

Três colunas — To Do, Doing e Done — com drag & drop via SortableJS para um fluxo de trabalho visual e intuitivo.

Vista Calendário

Visualize prazos e tarefas num calendário mensal interativo. Disponível no painel empresarial.

Estatísticas & Análise

Gráficos em tempo real com Chart.js — distribuição por estado, prioridade, tempo investido e produtividade da equipa.

Pomodoro Timer

Timer integrado por tarefa com registo automático de tempo. Sessões de 25 minutos para manter o foco.

Modo Deep Work

Ative o modo de concentração profunda: bloqueia notificações, escurece a interface e destaca apenas a tarefa atual.

Git Tools

Gere nomes de branches, mensagens de commit e checklists de code review diretamente a partir das tarefas.

Post-mortem

Crie relatórios de incidente com timeline, impacto, causa raiz e ações corretivas — aprenda com cada problema.

Tech Debt Tracker

Identifique e priorize dívida técnica. Classifique por severidade e gere tarefas de correção automaticamente.

Markdown & Código

Descrições em Markdown com syntax highlighting para JavaScript, Python, TypeScript, JSON e muito mais.

Dependências (DAG)

Defina tarefas bloqueadas por outras. O sistema mostra visualmente quais não podem avançar.

Automações

Regras "se isto então aquilo": mova tarefas, adicione labels ou receba notificações automaticamente.

Command Palette & Atalhos

Aceda a qualquer ação com Ctrl+K. Navegação rápida por teclado para utilizadores avançados.

Exportação

Exporte todas as tarefas em JSON ou CSV com um clique. Os seus dados ficam sempre acessíveis.

Gestão de Equipa

Painel empresarial com gestão de utilizadores, papéis (admin/membro), e organização centralizada.

PWA & Modo Offline

Instale como app nativa. Trabalhe sem internet — as alterações sincronizam quando voltar a estar online.
